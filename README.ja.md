## 使用方法
1. インストール  
   1. npm  
   `npm i @technote-space/github-action-pr-helper`
   1. yarn  
   `yarn add @technote-space/github-action-pr-helper`
1. 使用
```typescript
import { run } from '@technote-space/github-action-pr-helper';

run({
	actionName: 'Test Action',
	actionOwner: 'octocat',
	actionRepo: 'hello-world',
});
```
